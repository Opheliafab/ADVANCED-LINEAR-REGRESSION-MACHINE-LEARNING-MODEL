**PROJECT:**

IMPROVING ON LINEAR REGRESSION MODELS

**DESCRIPTION:**

This is series of codes showing how I improved on a regular basic regression model by adding Polynomial terms, interaction terms and regularization terms. During the course of these improvements, I also performed different experiments to ascertain which regression model best fit the data. 
After the application of each terms, I displayed the results of the of the models through visualization with various charts and graphs. 

Please note that the interface of this code shows that I wrote it as a part of the course requirements whilst undergoing the IBM Data Analytics with Excel and R programme.
